# Windows XP Portfolio

A nostalgic, fully interactive Windows XP-themed portfolio website built with React and TailwindCSS.

![Windows XP Portfolio](public/background.jpg)

## üåü Features

Experience a faithful recreation of the classic Windows XP interface right in your browser:

*   **Authentic Desktop**: Custom Bliss wallpaper with watermark, functional desktop icons, and taskbar.
*   **Login Flow**: Realistic User Selection and full-screen "Welcome" transition.
*   **Start Menu**: Fully functional Start Menu with pinned apps, "All Programs" submenu, and system shortcuts.
*   **Window Management**: Drag, minimize, maximize, and close windows just like the real OS.
*   **My Computer**: Browse drives like "Portfolio (C:)" and "Skills (D:)" with a sidebar task pane.
*   **Gallery (My Pictures)**: A folder view for photos with a custom "Picture and Fax Viewer" simulation.
*   **Command Prompt**: A working terminal simulation with custom commands (`skills`, `about`, `matrix`, `ping`).
*   **Project Explorer**: A unique way to browse my projects ("Modern Bar Website", "Gaming Website") with detailed stats.

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:5173` in your browser.

## üíª Custom Commands (cmd.exe)

Open the **Command Prompt** from the Start Menu or Desktop and try these:

*   `help` - List all available commands
*   `skills` - meaningful animation of technical skills
*   `about` - Short bio
*   `contact` - Display contact details
*   `matrix` - Enter the matrix (press any key to exit)
*   `color a` - Change terminal color to hacker green
